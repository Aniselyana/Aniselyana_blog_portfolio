# CSD 34203: Special Topics in Software Development
## Individual Project: GitHub Portfolio (Personal Blog Page)

### Project Overview
This repository contains my individual project portfolio, which is a personal blog page built as part of the CSD 34203 course assessment. The project demonstrates an understanding of the software development process, responsive web design, and basic interactivity.

---

### Features
* **Multi-page Layout:** Includes Home, About, Blog (with sample posts), and Contact pages.
* **Responsive Design:** Optimized for mobile, tablet, and desktop viewing using CSS Media Queries.
* **Interactive Theme Toggle:** Features a custom JavaScript button allowing users to switch between Light Mode and Dark Mode.
* **Clean Navigation:** Consistent header and footer implementation across all sub-pages.

---

### Technologies Used
* **HTML5:** For structural markup.
* **CSS3:** For styling, layouts, typography, and responsive design.
* **JavaScript (Vanilla JS):** For DOM manipulation and theme switching logic.
* **Git & GitHub:** For version control and project management.

---

### How to Run the Project Locally
1. Clone this repository to your computer or download the ZIP file.
2. Navigate to the project folder.
3. Open the `index.html` file using any modern web browser (e.g., Google Chrome, Microsoft Edge, Mozilla Firefox).
