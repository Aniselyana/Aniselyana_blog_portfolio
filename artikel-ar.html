AR IsyaratKu: Augmented Reality Application

*Proof of work: Real-time interaction exposure using Unity & Vuforia.*

This is an educational application that helps users learn Malaysian Sign Language (BIM). Powered by Unity and Vuforia technology, this application detects 3D hand movements to provide an interactive and effective learning experience.

This project includes research phases, 3D model development, and C# code integration to ensure accurate movement detection. The application has been tested with various lighting angles to ensure maximum usability for people with disabilities.
